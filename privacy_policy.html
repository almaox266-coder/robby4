นโยบายความเป็นส่วนตัว (Privacy Policy)

อัปเดตล่าสุด: 5 มีนาคม 2026 | Last updated: March 5, 2026

สรุปสั้นๆ: สกอร์ไทย ไม่เก็บ ไม่แชร์ และไม่ขายข้อมูลส่วนบุคคลของคุณ ข้อมูลทั้งหมดถูกจัดเก็บในเครื่องของคุณเท่านั้น

1. บทนำ

นโยบายความเป็นส่วนตัวฉบับนี้อธิบายวิธีที่ส่วนขยาย "สกอร์ไทย" (ScoreThai) สำหรับ Google Chrome จัดการเกี่ยวกับข้อมูลของผู้ใช้ เราให้ความสำคัญกับความเป็นส่วนตัวของคุณเป็นอย่างยิ่ง

2. ข้อมูลที่เราเก็บรวบรวม

เราไม่เก็บรวบรวมข้อมูลส่วนบุคคลใดๆ ส่วนขยายนี้จัดเก็บเฉพาะข้อมูลต่อไปนี้ ในเครื่องของคุณเท่านั้น (ผ่าน Chrome Storage API):

  • ทีมโปรด: รายชื่อทีมที่คุณเลือกติดตาม เพื่อส่งการแจ้งเตือน
  • การตั้งค่า: ค่ากำหนดการแจ้งเตือนและความถี่ในการรีเฟรชข้อมูล

ข้อมูลเหล่านี้ไม่ถูกส่งไปยังเซิร์ฟเวอร์ใดๆ และไม่ถูกแชร์กับบุคคลที่สาม

3. ข้อมูลที่เราไม่เก็บ

  • ชื่อ อีเมล หมายเลขโทรศัพท์ หรือข้อมูลระบุตัวตนใดๆ
  • ประวัติการท่องเว็บ
  • ข้อมูลตำแหน่งที่ตั้ง
  • คุกกี้หรือเทคโนโลยีติดตามอื่นๆ
  • ข้อมูลทางการเงิน

4. การเชื่อมต่อกับบริการภายนอก

ส่วนขยายนี้เชื่อมต่อกับ API ของบุคคลที่สาม (API-Football ผ่าน RapidAPI) เพื่อดึงข้อมูลผลบอลสดและราคาบอลเท่านั้น การเชื่อมต่อนี้:

  • ส่งเฉพาะคำขอข้อมูลฟุตบอล (เช่น ผลบอลสด ราคาบอล)
  • ไม่ส่งข้อมูลส่วนบุคคลของผู้ใช้ไปยัง API
  • ใช้การเชื่อมต่อแบบเข้ารหัส HTTPS เท่านั้น

5. สิทธิ์ที่ส่วนขยายใช้

สิทธิ์ (Permission) เหตุผลที่ใช้
alarms ตั้งเวลาดึงข้อมูลผลบอลเป็นระยะๆ
notifications ส่งการแจ้งเตือนเมื่อทีมโปรดทำประตูหรือก่อนแมตช์เริ่ม
storage บันทึกการตั้งค่าและทีมโปรดในเครื่องของคุณ
host_permissions เชื่อมต่อ API เพื่อดึงข้อมูลผลบอลและราคาบอล

6. ความปลอดภัยของข้อมูล

เนื่องจากข้อมูลทั้งหมดจัดเก็บในเครื่องของคุณเท่านั้น ความปลอดภัยของข้อมูลจึงอยู่ภายใต้ระบบรักษาความปลอดภัยของ Google Chrome และอุปกรณ์ของคุณ เราไม่มีเซิร์ฟเวอร์ที่จัดเก็บข้อมูลผู้ใช้

7. การเปลี่ยนแปลงนโยบาย

หากมีการเปลี่ยนแปลงนโยบายความเป็นส่วนตัวนี้ เราจะอัปเดตวันที่ด้านบนและแจ้งให้ทราบผ่านการอัปเดตส่วนขยาย แนะนำให้ตรวจสอบนโยบายนี้เป็นประจำ

8. ติดต่อเรา

หากคุณมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัวนี้ สามารถติดต่อเราได้ที่:

  • อีเมล: scorethai.ext@gmail.com

Privacy Policy (English)

Last updated: March 5, 2026

TL;DR: ScoreThai does NOT collect, share, or sell any personal data. All data is stored locally on your device only.

1. Introduction

This Privacy Policy explains how the "ScoreThai" (สกอร์ไทย) Chrome extension handles user data. We take your privacy very seriously.

2. Data We Collect

We do NOT collect any personal data. The extension stores only the following data locally on your device (via Chrome Storage API):

  • Favorite teams: Teams you choose to follow, used for notifications.
  • Settings: Your notification preferences and refresh interval.

This data is never sent to any server and is never shared with third parties.

3. Data We Do NOT Collect

  • Names, emails, phone numbers, or any personally identifiable information
  • Browsing history
  • Location data
  • Cookies or other tracking technologies
  • Financial information

4. Third-Party Services

The extension connects to a third-party API (API-Football via RapidAPI) solely to retrieve live football scores and odds data. This connection:

  • Only sends requests for football data (scores, odds)
  • Does NOT transmit any user personal information
  • Uses HTTPS encrypted connections only

5. Permissions Used

Permission Purpose
alarms Schedule periodic fetching of live scores
notifications Send alerts when favorite teams score or before matches start
storage Save settings and favorite teams locally
host_permissions Connect to the football data API

6. Data Security

Since all data is stored locally on your device, data security is managed by Google Chrome's security system and your device. We do not operate any servers that store user data.

7. Changes to This Policy

If this Privacy Policy is updated, we will update the date above and notify users through the extension update. We recommend reviewing this policy periodically.
